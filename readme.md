# Tradeious.com - B2B Marketplace Website

A modern, responsive e-commerce website for Tradeious, a B2B marketplace specializing in FMCG, tyres, electronics, and industrial supplies.

## 🚀 Features

✅ Fully responsive design (mobile, tablet, desktop)  
✅ Modern UI with smooth animations  
✅ Product categories with hover effects  
✅ Featured products with add-to-cart functionality  
✅ Shopping cart with localStorage persistence  
✅ Search modal with suggestions  
✅ Contact form with validation  
✅ Testimonials & trust indicators  
✅ Accessible (ARIA labels, keyboard navigation)  
✅ Optimized performance & SEO-ready
